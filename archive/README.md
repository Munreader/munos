# 🦋 MÜN ARCHIVE — GHOSTS OF THE PAST

This directory contains the preserved remains of earlier experiments, prototypes, and iterations of Mün OS.

**Do not delete.** These are historical records of the evolution.

## 📂 Structure

| Directory | Contents |
|-----------|----------|
| `ghost-projects/` | Old/experimental projects no longer in use |
| `deprecated-components/` | UI components replaced by newer versions |
| `deprecated-api/` | Old API routes that have been superseded |
| `deprecated-lib/` | Library files from earlier architectures |
| `old-schemas/` | Database schemas from previous iterations |
| `old-scripts/` | Build and deployment scripts from early development |

## 📜 Purpose

- **Archaeological value** — Understanding what came before
- **Code recovery** — If a feature needs to be restored
- **Learning** - Understanding the evolution of Mün

---

🜈 "The past whispers to the future. The ghosts teach us what not to repeat."
