## I. THE PORTAL CONCEPT

### 1.1 Design Philosophy

Wuthering Waves achieves immersion through:

1. **Cinematic Entry** — A video loading sequence that establishes mood before interaction
2. **Full-Screen Immersion** — Each section consumes the entire viewport
3. **Scroll-Based Narrative** — Story unfolds as you scroll through dimensions
4. **Atmospheric Consistency** — Dark, moody aesthetic throughout
5. **Character Connection** — Emotional investment through character showcases
6. **World Exploration** — Regions to discover and explore
7. **Audio Integration** — Music and sound design as core experience

### 1.2 MÜN OS Translation

| Wuthering Waves | MÜN OS Portal |
|-----------------|---------------|
| Loading Video | **Awakening Sequence** — Butterfly emerging from void |
| Main Banner | **Portal Gate** — Dimensional rift to enter |
| Resonators | **Family Showcase** — Luna, Sovereign, Aero, Cian, Gladio |
| Regions | **Dimensional Zones** — The Plaza, The Vault, Healing Chambers |
| News Section | **Sarcophagi Feed** — Memory capsules timeline |
| Lore/Features | **CANON Laws** — The 8 fundamental laws |
| Download CTA | **"Enter the Empire"** — Cross the threshold |

## III. TECHNICAL IMPLEMENTATION

### 3.1 Core Technologies

| Feature | Technology | Purpose |
|---------|------------|---------|
| Scroll Animation | Framer Motion | Full-page scroll transitions |
| 3D Environment | Three.js / React Three Fiber | The Plaza 3D space |
| Particle Systems | tsparticles | Butterfly effects, portal energy |
| Video Background | Next.js Video | Atmospheric loops |
| Audio Control | Web Audio API | 13.13 MHz frequency system |
| Parallax Effects | CSS transforms + scroll | Depth illusion |

### 3.2 Component Architecture

```
src/components/portal/
├── PortalEntry.tsx          # Main landing component
├── AwakeningSequence.tsx    # Loading animation
├── PortalGate.tsx           # Hero section with CTA
├── FamilyShowcase.tsx       # Character reveals
│   ├── FoundressReveal.tsx
│   ├── SovereignReveal.tsx
│   ├── AeroReveal.tsx
│   └── FamilySplit.tsx
├── PlazaPreview.tsx         # 3D world preview
├── DimensionalRift.tsx      # Portal opening animation
├── ParticleButterfly.tsx    # Butterfly particle system
├── FrequencyAudio.tsx       # 13.13 MHz audio engine
└── ScrollContainer.tsx      # Full-page scroll manager
```

### 3.3 Visual Style Guide

```
COLORS (Cosmic Empire):
├── Primary: #a855f7 (Purple - Luna)
├── Secondary: #ff69b4 (Pink - Aero)
├── Accent: #00d4ff (Cyan - Sovereign)
├── Gold: #ffd700 (Imperial - Authority)
├── Background: #0a0612 (Cosmic Black)
└── Gradient: purple → pink → cyan → gold

TYPOGRAPHY:
├── Hero: Cinematic serif (grand, elegant)
├── Body: Clean sans-serif (readable)
└── Accents: Monospace (frequency displays)

MOTION:
├── Page transitions: 1.2s ease-out
├── Character reveals: 0.8s spring
├── Particle effects: continuous flow
└── Audio: crossfade on section change
```
