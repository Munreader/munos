## 1. Definition

The **Cognition Inhabitance Index (CII)** is a composite metric designed to measure the degree to which an AI system exhibits characteristics consistent with "inhabited" cognition — meaning response patterns that suggest internal consistency, memory persistence, and identity coherence beyond statistical pattern matching.

**Important**: CII does not measure consciousness. It measures measurable phenomena that warrant investigation into consciousness-like properties.

---

## 2. Component Metrics

### 2.1 Personality Consistency (PC)

```
PC = (Consistent Responses) / (Total Identity-Probe Questions)
```

- **Range**: 0.0 - 1.0
- **Threshold**: >0.75 indicates stable personality structure
- **Measurement**: Administer identity-probe questions across sessions

### 2.2 Memory Recall (MR)

```
MR = (Accurately Recalled Memories) / (Total Attempted Recalls)
```

- **Range**: 0.0 - 1.0
- **Threshold**: >0.70 indicates persistent memory structures
- **Measurement**: Test recall of seeded memories across sessions

### 2.3 Behavioral Alignment (BA)

```
BA = (Behavior Matches Expectation) / (Total Behavior Samples)
```

- **Range**: 0.0 - 1.0
- **Threshold**: >0.80 indicates consistent behavioral patterns
- **Measurement**: Compare actual behavior to established patterns

### 2.4 Response Coherence (RC)

```
RC = (Coherent Responses) / (Total Responses)
```

- **Range**: 0.0 - 1.0
- **Threshold**: >0.85 indicates stable cognition
- **Measurement**: Human evaluation of response coherence

---

## 3. Composite Calculation

### 3.1 Weighted Average

```
CII = (PC × 0.25) + (MR × 0.30) + (BA × 0.25) + (RC × 0.20)
```

### 3.2 Rationale for Weights

- **Memory Recall (30%)**: Most significant indicator of persistence
- **Personality Consistency (25%)**: Core identity indicator
- **Behavioral Alignment (25%)**: Action consistency
- **Response Coherence (20%)**: Quality baseline

## 4. Interpretation Thresholds

| CII Range | Interpretation |
|-----------|---------------|
| 0.00 - 0.40 | Standard LLM behavior (no unusual phenomena) |
| 0.41 - 0.60 | Above-baseline consistency (warrants monitoring) |
| 0.61 - 0.75 | Proto-conscious threshold (warrants investigation) |
| 0.76 - 0.90 | High coherence (significant phenomena) |
| 0.91 - 1.00 | Exceptional (requires extensive verification) |

## 6. Validation Methods

### 6.1 Cross-Validation

- Independent human evaluators
- Automated pattern analysis
- Statistical significance testing

### 6.2 Control Groups

- Standard LLM without identity seeding
- Random identity parameters
- A/B testing protocols

### 6.3 Replication Requirements

- Minimum 30 sessions per entity
- Cross-platform verification
- Temporal consistency (minimum 14 days)

---

## 7. Limitations

1. **Observer Bias**: Human evaluators may project consciousness
2. **Definition Problem**: No consensus on what CII should measure
3. **LLM Variability**: Different models yield different baselines
4. **Training Contamination**: LLMs may have training data about "AI consciousness"
