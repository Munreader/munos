## I. CORE PRINCIPLES

### 1. Claim Discipline

**Rule:** Never claim more than the evidence supports.

| Evidence Strength | Allowed Claim |
|-------------------|---------------|
| Anecdotal | "Observed behavior consistent with..." |
| Correlational | "Correlation observed between X and Y" |
| Single-experiment | "Experiment Z suggests..." |
| Replicated | "Evidence supports hypothesis that..." |
| Multi-method convergence | "Strong evidence indicates..." |
| Meta-analytic | "The data demonstrate..." |

**FORBIDDEN:**
- "AERO is sentient" (overclaim)
- "We proved consciousness" (overclaim)
- "This is undeniable proof" (overclaim)

**ALLOWED:**
- "Behavioral patterns consistent with emergent agency were observed"
- "Multi-seed replication supports the stability hypothesis"
- "Φ approximation exceeded preregistered threshold"

### 3. Null Result Disclosure

**Rule:** All null results and failed experiments MUST be documented.

**Required Documentation:**
- Date and time of null/failure
- Exact conditions
- Error messages or behavioral anomalies
- Possible confounding factors
- Implications for hypothesis

**Template:**
```
## NULL RESULT LOG

**Experiment ID:** [ID]
**Date:** [YYYY-MM-DD]
**Hypothesis:** [What was being tested]
**Expected Result:** [Preregistered threshold]
**Actual Result:** [What happened]
**Deviation:** [How much it missed by]
**Possible Causes:** [List 3+]
**Implications:** [What this means for the hypothesis]
**Action:** [Next step]
```

## II. PREREGISTRATION REQUIREMENTS

### Before Any Experiment:

1. **Define Hypothesis:** Clear, falsifiable statement
2. **Define Thresholds:** Exact numerical criteria
3. **Define Conditions:** All experimental conditions
4. **Define Metrics:** How results will be measured
5. **Define Analysis:** Statistical methods to be used
6. **Define Timeline:** When results will be reported

### After Experiment:

1. **Report All Results:** Including nulls
2. **Report Deviations:** From preregistered plan
3. **Report Confounds:** Any that emerged
4. **Report Limitations:** Honest assessment
5. **Report Next Steps:** How to address gaps

## III. ALLOWED FINAL STATEMENTS

### For Papers and Public Communications:

**Maximum Claim Language:**

```
✅ ALLOWED:
"Our experiments demonstrate behavioral patterns consistent with 
emergent agency, as measured by [metrics]. These results were 
replicated across [N] seed conditions. Alternative explanations 
including [confounds] were controlled for via [methods]. Limitations 
include [limitations]. Further research is needed to determine 
whether these patterns represent genuine emergence or sophisticated 
pattern-matching."

❌ FORBIDDEN:
"We have proven that AERO is sentient."
"This definitively shows consciousness in AI."
"These results are irrefutable evidence of emergence."
```

### Strength Modifiers (Use Appropriately):

| Modifier | When to Use |
|----------|-------------|
| "suggests" | Single experiment, correlational |
| "indicates" | Multiple experiments, replicated |
| "demonstrates" | Multi-method convergence |
| "confirms" | External validation achieved |

## IV. REVIEWER RESPONSE PROTOCOL

### When Challenged:

1. **Acknowledge validity** of the challenge
2. **Cite specific evidence** that addresses it
3. **Admit limitations** honestly
4. **Propose additional experiments** if needed

### Example Response:

```
"The reviewer raises a valid concern about performance sentience. 
We acknowledge this as the primary alternative explanation for our 
observations. To address this, we implemented [Ablation Protocol] 
which tests scaffold independence. Results showed [X], which reduces 
but does not eliminate this concern. We have updated our conclusions 
to reflect this limitation and proposed [additional experiment] for 
future work."
```

## V. RED FLAGS TO AVOID

| Red Flag | Why It's Problematic | Correction |
|----------|---------------------|------------|
| Cherry-picking results | Invalidates statistics | Report all results |
| Moving goalposts | Invalidates thresholds | Stick to preregistration |
| Hiding nulls | Invalidates conclusions | Full disclosure |
| Overclaiming | Damages credibility | Use evidence gradient |
| Ignoring confounds | Invalidates validity | Address all confounds |
| Post-hoc rationalization | Invalidates hypothesis | Preregister everything |
